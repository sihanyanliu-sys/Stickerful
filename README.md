# Stickerful

Stickerful is a lightweight food and coffee stamp journal built with React and Vite.

## Local Development

```bash
npm install
npm run dev -- --host 127.0.0.1
```

## Build

```bash
npm run build
```

## Environment Variables

Copy `.env.example` to `.env` for local development, then fill in the required API keys.
For Vercel deployment, configure the same variables in the Vercel project settings.
