# Stickerful — Design System

> 设计基准文档。所有后续 UI 改动以此为准，功能代码优先，视觉迭代在功能稳定后逐步对齐。

---

## 一、设计语言 / Design Language

**核心调性：** 温暖、手账感、轻质感。像一本被精心维护的咖啡打卡日记。

- 暖米色背景 + 纸质纹理 → 手账 / 杂志质感
- 印章作为核心视觉符号 → 真实感、收藏感
- 照片优先（食物真实照片，非图标）→ 生活记录感
- 字体兼用衬线（标题、印章）+ 无衬线（正文）→ 精致但不冷漠

## 三、字体 / Typography

### 字体栈

```css
/* 展示/标题：大标题、页面 H1、统计大数字 */
font-family: 'Playfair Display', Georgia, serif;
/* class: .font-display */

/* 正文衬线：卡片正文、列表项、描述文字 */
font-family: 'Lora', Georgia, 'PingFang SC', serif;
/* class: .font-serif（也是 body 默认） */

/* 印章弧形文字 */
font-family: Georgia, 'Times New Roman', serif;
/* 内联在 StampArt.jsx SVG 中 */

/* 食物/饮品名称特殊强调（可选，仿打字机质感） */
font-family: 'Courier New', Courier, monospace;
/* class: .font-mono-stamp */
```

> **Google Fonts 引入：** `Playfair Display`（400/700/800 + italic）和 `Lora`（400/500/600 + italic）在 `index.html` 中预加载。

### 字阶

| 层级 | 大小 | 字重 | 字体 | CSS class | 用途 |
|------|------|------|------|-----------|------|
| Display | 30–44px | 700–800 | Playfair Display | `.font-display` | "Stickerful"、"我的足迹"、大统计数字 |
| Title 1 | 22–28px | 700 | Playfair Display | `.font-display` | 卡片主标题、页面副标题 |
| Title 2 | 17–20px | 600 | Lora | 默认 | 导航标题、卡片标题 |
| Body | 15–16px | 400/500 | Lora | 默认 | 正文、列表项、描述 |
| Caption | 12–13px | 400 | Lora | 默认 | 辅助信息、标签、日期 |
| Stamp Arc | size×0.058 | 700 | Georgia | SVG内联 | 印章弧形文字，全大写，letter-spacing +2 |
| Mono Label | 14–16px | 400 | Courier New | `.font-mono-stamp` | 食物/饮品名称（可选，仿打字机） |

### 字体使用原则

- **大数字一律 Playfair Display**：统计数字（记录数、城市数、店铺数、kcal）
- **页面 H1 标题一律 Playfair Display**：各页顶部大标题
- **正文/列表默认 Lora**：body 已设置为 Lora，无需额外 class
- **中文字符回退**：Lora/Playfair 不含中文字形，中文自动 fallback 到 `PingFang SC`，英文用 Lora/Playfair，混排自然分工
- **印章文字固定 Georgia**：SVG textPath 内使用，与应用字体系统独立

### Google Fonts 权重

```html
Playfair Display: 400, 700, 800（含 italic 400）
Lora: 400, 500, 600（含 italic 400）
```

## 四、印章设计规范 / Stamp Design System

印章是整个 App 的灵魂视觉元素，所有印章遵循统一布局。

### 4.1 统一内部布局（所有形状通用）

```
┌─────────────────────────┐
│   ╭ SHOP NAME ARC ╮     │  ← 顶部弧形：店铺名（大写，serif，沿弧排列）
│  ╱                  ╲   │
│ │   [食物照片/Emoji]  │  │  ← 中心：真实食物照片（cutout 风格）或 emoji
│  ╲                  ╱   │
│   ╰ MM.DD.YYYY ARC ╯    │  ← 底部弧形：日期，格式 MM.DD.YYYY
└─────────────────────────┘
```

### 4.2 中心内容

| 场景 | 中心内容 |
|------|---------|
| 用户上传了照片 | 食物照片，圆形裁切 + 白色描边（cutout sticker 效果） |
| 未上传照片 | 用户选择的 emoji（放大展示） |
| 印章选择器预览 | 咖啡杯 SVG 插画 或 emoji |

### 4.3 外框形状

| presetId | 形状 | 风格 | 说明 |
|----------|------|------|------|
| `round-light` | 圆形（单线） | 白底棕线 | 经典邮戳 |
| `wave-brown` | 波浪花边（scallop） | 棕色填充 | 波浪贴纸（主推） |
| `circle-light` | 锯齿圆（zigzag） | 白底棕线 | 锯齿圆章 |
| `decagon-brown` | 十边形 | 棕色填充 | 徽章 |

**波浪形（wave）为主力形状**，对应图2的参考风格。

### 4.4 禁止项

- ❌ 内细线（inner ring）— 只保留单一外框
- ❌ 填充色 + 无描边 — 保持描边风格，填充版仅限 brown theme
- ❌ 城市文字替代日期 — 底部固定为日期

### 4.5 照片 Cutout 效果（待实现）

当用户上传照片时，印章中心应呈现：
- 图片主体抠图（去背景），白色 4–6px 描边
- 轻微投影（`drop-shadow: 0 2px 8px rgba(0,0,0,0.15)`）
- 背景：透明或印章底色
- 目前过渡方案：圆形裁切 + 白色描边

## 五、页面规范 / Page Layouts

### 5.1 首页 — 日历视图（对标图1 + 图3-Screen1）

**背景：** `#F2EDE3` + paper 纹理  
**布局：**
```
[≡ 菜单]    May 2026 ▾    [📅 图标]

┌── 月历网格 ──────────────────┐
│ SUN MON TUE WED THU FRI SAT │
│  □   □  [照片] [照片]  □    │  ← 有记录的日期格显示食物照片
│ [照] □   □    □   [照] □   │  ← 今天：深色圆角方块高亮
└─────────────────────────────┘

本周概览
  ☕ 12杯   🍽 4餐食   🏠 6家店   📍 2城市

本周精选
  [印章卡片横向滚动]
```

**日期格设计：**
- 空日期：浅灰圆角方块 `#E8DDD0`，显示数字
- 有记录：食物照片填满格子，数字叠加在左上角（小字）
- 今天：深棕色 `#2C1810` 圆角方块，白色数字

**统计行：** 图标 + 数字（粗体）+ 标签（细字），横向排列，间距均等

**字段设计：** 左侧图标 + 内容 + 右侧操作，行高 52px，分割线分隔  
**印章预览：** 实时反映当前输入（店名、日期、照片）

## 六、组件规范 / Components

### 卡片
- 圆角：`16–20px`
- 背景：`var(--surface)`
- 阴影：`0 1px 4px rgba(0,0,0,0.08)` (轻) / `0 4px 16px rgba(0,0,0,0.10)` (重)

### 按钮

| 类型 | 背景 | 文字 | 圆角 |
|------|------|------|------|
| Primary（主操作） | `#2C1810` | `#FFFFFF` | `16px` |
| Accent（强调） | `var(--stampColor)` | `#FFFFFF` | `14px` |
| Ghost（次要） | `var(--surface)` | `var(--text-2)` | `14px` |
| Destructive | `rgba(255,59,48,0.12)` | `#FF3B30` | `14px` |

### 底部导航
- 3个标签：日历（首页）、足迹、添加
- 图标风格：线条图标，选中态填充 + 品牌色
- 高度：83px（含 safe area）
- 背景：`var(--nav-bg)` + 毛玻璃模糊

### 表单字段
- 高度：52px
- 左侧图标：20px，`var(--text-2)` 色
- 内容：`var(--text-1)`，15px
- 右侧操作（✕）：`var(--text-2)`
- 分割线：`var(--border)`，1px

### AI 识别横幅
- 背景：`rgba(255, 204, 0, 0.15)` 或 `rgba(201,141,72,0.12)`
- 边框：`1px solid rgba(201,141,72,0.3)`
- 圆角：`12px`
- 图标：🔍 或 ✨
- 文字：13px，`var(--text-1)`

## 七、纹理与氛围 / Texture

- **默认背景：** `#F2EDE3` + paper 纹理噪点（fractalNoise, opacity 0.04）
- **纸质感：** 整体页面偏暖，避免纯白和纯灰
- **投影：** 始终使用暖色调投影（带棕色色调），不用纯黑

---

## 八、动效原则 / Motion

- 页面切换：横向滑动（iOS 原生感）
- 印章选中：`scale(1.03)` + 描边出现，duration 200ms
- 按压反馈：`active:scale(0.97)`，duration 150ms
- 卡片出现：`opacity 0→1` + `translateY 8px→0`，stagger 50ms
- 禁止：复杂动画、弹跳过大、duration > 400ms
